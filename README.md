## ✨ Features

- **🎮 Online Multiplayer**: Real-time room-based multiplayer powered by Firebase Realtime Database. Play with your friends instantly across the globe.
- **📱 Pass N Play**: Gather around a single device and play locally with your friends or against computer opponents.
- **💎 Stunning Glassmorphism UI**: A completely overhauled, modern user interface featuring frosted glass panels (`BackdropFilter`) over vibrant, animated gradient backgrounds.
- **🧊 3D Glossy Graphics**: All game pieces, the board, and bases feature beautiful multi-stop radial and linear gradients to simulate physical, glossy plastic pieces.
- **🎲 Real 3D Dice Physics**: Throw the dice and watch it tumble with a real bouncy 3D rotation (`Matrix4`) rather than just a flat image.
- **🛠 Fully Interactive Menus**: Dynamic Bottom Navigation Bar, Player Profile dialogs, Settings bottom sheets, and daily reward popups.

## 🚀 Getting Started

### Prerequisites

Make sure you have the following installed:
- [Flutter SDK](https://flutter.dev/docs/get-started/install)
- Android Studio / VS Code
- A connected Firebase Project

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/CodeWithBasu/Ludo-App.git
   ```

2. Navigate into the project directory:
   ```bash
   cd Ludo-App
   ```

3. Install dependencies:
   ```bash
   flutter pub get
   ```

4. Run the app:
   ```bash
   flutter run
   ```

## 🏗️ Tech Stack

- **Framework:** Flutter / Dart
- **Backend/Database:** Firebase Realtime Database
- **UI/UX Design:** Custom Painters, Animated Builders, 3D Matrix Transforms

## 🎨 Design Philosophy
The aesthetic goal of Ludo Master was to create a **"Vibrant & Clean 3D"** look. Rather than relying on flat material colors, the UI heavily utilizes CSS-like gradients, inset shadows, and frosted glass to mimic the premium, glossy feel of a high-end Nintendo or Mario Party game.
